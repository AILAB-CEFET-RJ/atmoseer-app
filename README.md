# Atmoseer-app-frontend

## Get started

Clone the repository
```sh
git clone https://github.com/rhyanpinto/atmoseer-app-frontend.git
cd atmoseer-app-frontend
```

Initialize atmoseer project
```sh
git submodule update --init --progress
```
